## 运行项目

**安装依赖:**

*npm*

```bash
npm i
```

*yarn*

```bash
 yarn install
```





**启动项目:**

*npm*

```bash
npm run dev
```

*yarn*

```bash
yarn run dev
```





**项目配置:**

请参考config目录下的配置文件
